# 01 — Brand Identity

**Priority:** REQUIRED. Nothing else can be personalized credibly without this.

## What to upload here

- [ ] Full brand guidelines PDF or InDesign export (the "master" doc)
- [ ] Brand standards manual / visual identity guide
- [ ] Color specification sheets (hex, RGB, CMYK, Pantone)
- [ ] Tagline, mission statement, positioning
- [ ] Brand architecture diagrams (parent brand, sub-brands, project brands like Rio Grande LNG, CP2, etc.)
- [ ] Verbal identity / tone of voice summary (if standalone)
- [ ] Any "do and don't" visual examples

## File types expected

`.pdf`, `.indd`, `.ai`, `.sketch`, `.fig`, `.docx`, `.pptx`, `.png/jpg` (of guide pages)

## What Claude will extract

- Primary / secondary / tertiary color palettes with all color systems (hex, RGB, CMYK, Pantone)
- Color usage ratios (60/30/10 rules, accent vs. dominant)
- Tagline exact wording and usage rules
- Brand architecture: how parent brand relates to project brands (NDLNG vs. Rio Grande LNG vs. CP2 vs. Next Carbon Solutions)
- Minimum sizes, clear space formulas, safe zones
- Which colors can touch each other, which cannot
- Co-branding rules (when partner logos appear — TotalEnergies, customers, etc.)

## Feeds these skills

- `skills/brand-guidelines/` (full rewrite)
- `skills/pptx/` (default theme)
- `skills/docx/` (default fonts/colors)
- All new skills (inherit brand as baseline)

## Notes / gaps to flag back to you

If any of these aren't in your uploads, Claude will flag as gaps:
- Dark-mode / reversed brand variants
- Digital vs. print palette differences
- Accessibility-compliant color pairs (WCAG AA/AAA)
- Historical / deprecated brand elements (important — "don't use this anymore")
